Total Fertility Rate (Live births per woman) in 1960-1965!


I have used the Total Fertility Rate from the UN Data website. The dataset consists of countries and their fertility rates over 5 years. I have implemented the following functionalities-

  • Sorted countries alphabetically.
  • Showed all 10 countries in current sorting order.
  • Showed top 5 countries by value by keeping current sorting order.
  • Showed bottom 5 countries by value, keeping current sorting order.
  • Sorted countries alphabetically.
  • Sorted in ascending order by value
  • Sorted in descending order by value
